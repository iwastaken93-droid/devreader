## 2024-05-24 - Inline Feedback for Clipboard Actions
**Learning:** In the absence of a global toast notification system, users lack feedback when performing clipboard actions (like copying a code snippet), leading to confusion about whether the action succeeded.
**Action:** Implemented an inline visual feedback pattern where the "Copy" icon temporarily switches to a green "Check" icon for 2 seconds before reverting. This provides clear, immediate feedback without requiring a full toast system implementation.
